## Creating a virtual env
- Install virtualenv package
  ```bash
  pip3 install virtualenv
  ```
- Create a virtualenv named env
  ```bash
  python3 -m venv env
  ```
- Activate
  ```bash
  source env/bin/activate
  ```
- Deactivating
  ```bash
  deactivate
  ```
- listing installed package
  ```bash
  pip freeze
  ```
